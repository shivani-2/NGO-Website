# NGO-Website

Problem Statement:
The main aim of this project is to give back to society by volunteering or by donating.

The user can register to our NGO, either to donate or to volunteer.  For this, the user will fill a form and enter their full name, birth-date, email and contact number. You can volunteer at numerous events conducted by the various chapters of this NGO, by registering as a volunteer on this portal. Every user will also have their own dashboard displaying all their information, the chapters they are part of, their post in the NGO committee and the events they volunteered in. 
It informs the user when and where the event is conducted, i.e the date, time and location of the event is stored. We are also providing the user with a comprehensive description of each event that is being conducted by that respective chapter. We are also providing you with the address as to where exactly it is conducted. We are offering you two types of payment services, online and offline (online for those who are unable to be there in person).  
It is a platform for you to donate money, clothes, books, food and stationery to the needy. The user can apply for the position of chapter-head, secretary, treasurer, core-member, member or a volunteer. The range of access to the website depends on whether the person is a citizen or a part of the organization.

Features:

1) Individual login system
2) Age is being asked when the user is registered, such that only users above the age of 18 can donate.
3) The user(above the age of 18) will be able to select the number of clothes, books, food packets and stationery they want to donate.
4) Different modes of payment (online and offline).
5) Admin can add events(Its description and date of conduction).
6) Every event is conducted by a chapter, which provides the user with a detailed address as to where that event is conducted.
